# Tailwind Formatter

---

## O que faz?

Seleciona um texto como:

```tsx
"mx-auto my-8 flex items-center";
```

Clica com o botão direito → **"Formatar Tailwind"**

E ele vira:

```tsx
"mx-auto", "my-8", "flex", "items-center";
```
